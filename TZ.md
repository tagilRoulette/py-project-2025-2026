Техническое задание на проект: Pomodoro Timer Analyzer
1. Цель проекта: Проект позволяет использовать технику тайм-менджмента "Метод помидора". Данные о потраченном времени собираются и для проведения анализа о продуктивных часах пользователя. Есть возможность экспорта данных в Google Sheets.
2. Роли пользователей:
    Гость - использование таймера Помодоро;
    Зарегистрированный пользователь - сохранение настроек для таймера, сохранение данных о времени продуктивности, экспорт данных в Google Sheets.
4. Модели данных:
    Модель 1: User (Пользователь): пользователи сайта. Поля: name (PK), email (unique), password, is_active.
    Модель 2: Sheet (Лист): таблицы Google Sheets. Поля: owner (PK, FK-User), url (unique).
    Модель 3: PomodoroTimings: сохраненные для каждого пользователя тайминги техники. Поля: owner (PK, FK-User), work_time, break_time, long_break_time.
    Модель 4: ActivityTime (Время активности): данные об активности зарег. пользователей. Поля: owner (PK, FK-User), time_span, activity_type (PK, FK-Activity), date
    Модель 5: Activity (Активность): типы активности. Поля: ID (PK), name.
6. Ключевой функционал:
* Пользователи и гости могут запускать таймер Помодоро.
* Пользователи могут менять тайминги таймера.
* Пользователи могут смотреть статистику по своему времени продуктивности, экспортировать в Google Sheets.
5. Внешние интеграции / Аналитика: Google API, pandas.
